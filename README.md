## Usage

Start Chat Room Server on localhost
```sh
java -cp demo-1.0-RELEASE.jar edu.gwu.cs6431.multichat.demo.server.DemoServer
```

Start Chat Room Client
```sh
java -jar demo-1.0-RELEASE.jar
```
